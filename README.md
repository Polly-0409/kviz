# Квиз аппаратной косметологии для тела

Современный адаптивный квиз для подбора процедур аппаратной косметологии с персональной рекомендацией и комбинированными протоколами.

## Что умеет проект

- Многошаговый квиз с валидацией, прогрессом и стартовым экраном.
- Расчет персональной рекомендации по ответам клиента.
- Поддержка комбинированного курса (2+ процедуры).
- Расчет количества процедур, частоты и ориентировочных сроков.
- Экран результата с показаниями, ограничениями и ожидаемым эффектом.
- Мультивыбор проблемных зон для более точного подбора.
- Обновленный стильный интерфейс (hero-блок, декоративный фон, glassmorphism-карточки, плавные hover/focus состояния).

## Сценарий использования

1. Клиент проходит расширенную анкету (цели, зоны, состояние, тонус, отечность, формат курса и т.д.).
2. Алгоритм считает score для процедур.
3. Показывается:
   - основная рекомендация;
   - альтернативы;
   - комбинированный план (если релевантно);
   - длительность и частота курса.

## Основные процедуры в логике подбора

- LPG - вакуумно-роликовый массаж
- Кавитация
- Эндосфера - виброкомпрессионный массаж
- RF-лифтинг тела
- Горячий вакуум
- Биофатон
- Фотоомоложение
- Ручной массаж

## Важно
